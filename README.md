## Overview
This project demonstrates how to host a static website using **Amazon S3** within the **AWS Cloud** infrastructure.

AWS Cloud Computing provides on-demand access to computing power, storage, and databases, allowing users to pay only for what they use.

---

## Architecture
![AWS S3 Architecture](./img/architecture.png)

**Flow:**
- Users access the website through a web portal
- Static files are stored in an Amazon S3 bucket
- Access is managed using a Bucket Policy

---

## Key Features
- Amazon S3 bucket creation and configuration
- Bucket policy for secure access
- Static website hosting enabled
- Hosting of HTML, CSS, JavaScript, and image files
- Public access via S3 website endpoint

---

## Concepts Covered
- Amazon S3 (Simple Storage Service)
- Buckets and objects
- AWS Regions
- Access policies (JSON)
- Static website hosting
- Scalability and high availability

---

## Project Goal
This project was created for learning purposes to understand the fundamentals of AWS Cloud and static website hosting using Amazon S3.
